## 🌟 Overview

**Suzume Game** is a story-driven adventure experience inspired by the themes of mystery, exploration, and emotional storytelling from *Suzume no Tojimari*. Players journey through beautifully crafted environments, solve puzzles, unlock hidden secrets, and restore balance between worlds.

The game focuses on:

- 🗺️ Exploration
- 🧩 Puzzle Solving
- 🎭 Storytelling
- ⚔️ Adventure Gameplay
- 🌌 Fantasy Environments

## 🚀 Getting Started

### Prerequisites

Before running the project, make sure you have:

- Unity Hub
- Unity Editor (2022+ Recommended)
- Git

### Clone Repository

```bash
git clone https://github.com/kriss2012/Suzume-Game.git
```

### Open Project

1. Open Unity Hub
2. Click **Open Project**
3. Select the cloned repository
4. Wait for Unity to import assets
5. Open the main scene
6. Press ▶ Play
